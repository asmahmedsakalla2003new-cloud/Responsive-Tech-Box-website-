# TecBox Incubator Website

## Overview
TecBox is an educational business incubator platform designed to support and empower innovators and startups in the education field. The platform showcases projects, training programs, events, and provides essential resources for students and graduates.

> ⚠️ **Note:** The website content and interface are implemented in **Arabic**.

---

## Features
- **Homepage:** Displays latest news, featured events, and counters showing incubator statistics.
- **About Us:** Information about the incubator, its mission, and team.
- **Training Programs:** Details of educational and training programs offered by the incubator.
- **Events & Media Center:** Highlights past and upcoming events, media coverage, and announcements.
- **Contact Us:** Provides contact details, location, and working hours.
- **Responsive Design:** Fully responsive layout compatible with desktop and mobile devices.
- **Carousel & Cards:** Dynamic carousel for announcements and card-based sections for news and events.
- **Footer with Social Links:** Includes social media buttons and contact information.
- **News Ticker / Slider:** Displays important announcements at the bottom of the page.

---

## Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** None (static website)
- **Icons & Fonts:** Font Awesome
- **Version Control:** Git

---

## Usage

- **Navigate the website using the top navbar.**
- **Click on news cards to read more details.**
- **Use the "Join Us" button to access registration forms for programs.**
- **Follow social media buttons in the footer for updates.**
